Numpy basics

Posted on 08-09-2019 Edited on 09-09-2019 Comments:

  1. import numpy as np

  2. The data type in numpy is called array

  3. a = np.array([1,2,3]) to create a new array

  4. a=np.arange(1,12,3) from 1 to 11 with a step of 3 i.e. [1,4,7,10] Super useful in matplotlib

    The starting value will be included for sure while the ending value may not be included.

  5. a=np.linspace(1,13,5) create an array with 5 values and the step between them will be determined accordingly i.e.[1,4,7,10,13] Both, beginning and ending values will be included.

  6. a = a.reshape(3,2) change the dimension of an existing array. It means 3 rows and 2 columns here(2d array).

​ We may also create a multi-dimensional array manually using a = np.array([(1,2),(3,4),(5,6)])

  1. print(a.shape) will give us the shape of the array.

  2. print(a.size) will give us the length of the array kinda similar to len(list1)

  3. print(a.dtype) will give us the datatype of elements inside the array.

  4. print(a>1) to test the size of the array. This will automatically compare all elements inside the array with 1 and return True or False for each element.

  5. a = a*3 will multiply every element in array by 3. Other Math operations can also be applied this way.

  6. a = np.zeros((3,4)) This creates a 3*4 array with zeros. Similarly, we have a = np.ones((1,10))

  7. a = np.random.random((3,4)) create a 3*4 array with random decimal values under 1.

  8. a = np.random.randint(0,10,5) create an array with 5 elements random integers ranging from 0-10

    • print(a.sum()) will sum up all elements inside the array.
    • print(a.sum(axis=1)) sum up elements in each row (This axis trick also works for min, max, mean and std below)
    • print(a.sum(axis=0)) sum up elements in each column

  10. print(a.min()) will give the minimum element.

    print(a.max()) will give the maximum element.

  11. print(a.mean()) will give the mean of the array.

  12. print(a.std()) will give the standard deviation of the array.

  13. value = np.random.choice(a) will help you choose a random value from the array a.

  14. print(a.ndim) to find the dimension of the array.

  15. slicing of the array a = np.array([(1,2,3),(4,5,6)])

    If we want 3 , print(a[0,2])

    If we want 3 & 6, use print(a[0:,2]) This is equivalent to print(a[0:2,2])
