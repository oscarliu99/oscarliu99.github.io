HTML basics

Posted on 08-09-2019 Edited on 11-09-2019 In Programming Comments:

1. Tags

DOCTYPE

The tag as shown above will tell the internet explorer that we are writing an html file. For this tag, we don’t need an ending tag and for other tags(container tags) we need an ending tag for each starting tag.

html

Here are html tags with a starting tag and an ending tag. / is used to denote that it is an ending tag. This is the highest level tag in our website, other contents are under these tags.

head

Another type of tags are head tags. We can define the title, description of the document and also other information related to the document. It is like the settings of the html file.

Python revision notes

Posted on 08-09-2019 Edited on 12-09-2019 In Programming Comments:

  1. Switch data in two variables: a,b = b,a This is something unique to Python.

  2. print(variable, end = "") If we don’t want to skip a line after printing variable. The default value for end attribute is \n(skip a line).

  3. When we use comparison operators in Python(>, < etc.). It will always return a Boolean value(True, False)

  4. print(1==True);print(0==False) These two statements will always return True. As in Python 1 also means True and 0 also means False.

  5. print(0==0.0) will also return True even though they are different data types.

  6. There is a special operation symbol which is in. For example, print(“a” in “abc”). This will return True.

  7. 3 types of logic order in Python: plain sequence(顺序), loop(循环) and branches(分支).

  8. while True will produce an infinite loop.

Mysql basics

Posted on 08-09-2019 Edited on 11-09-2019 In Programming Comments:

  1. Database(DB): Any collection of related information that can be stored in different ways.

  2. Database management system(DBMS): software used to manage data in the database.

  3. C.R.U.D: Create Read Update Delete

  4. Relational databases(SQL): organize data onto one or more tables similar to Excel.

  5. Non-relational databases(no SQL): Everything but not SQL like JSON,XML etc.

  6. SQL(Structured Query Language): standard language interacts with DBMS to perform C.R.U.D operations.

  7. Database queries: requests made to the DBMS for specific information, in other words, asks the database for specific pieces of information

  8. There is always a primary key to a table, each row has its own primary key.
