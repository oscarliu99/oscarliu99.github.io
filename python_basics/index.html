Python basics

Posted on 08-09-2019 Edited on 09-09-2019 In Programming Comments:

  1. \n will create a new line in a string. print("apple\npear")

  2. string.lower() will change every character in a string to lower cases.

  3. string.upper() will change every character in a string to upper cases.

  4. len(string) will return the length of the string.

  5. string[3] will could index a string(start from zero)

  6. string.index(A) will return the position where A first time appear in the string. If A is consisted of multiple characters, it will return the position of the first character in A. If A can’t be found in the string, you will get an error

  7. string.find(A) very similar to .index(A). The difference is that if we couldn’t find A in the string, it will return -1 instead of an error.

  8. string.replace(old,new,times) The old parameter inside the string will be replaced by the new parameter. Without specify times, all old substring will be replaced by new. We may only want to replace old substring certain number of times, simply fill in the times parameter.

  9. * means multiply in Python. ** means exponential. We can also use pow(A,B) which stands for A to the power of B.

  10. % will return the remainder of the division.

  11. 1// means division and round down the answer. Remainder: print(5//2) will return 2 while print(-5//2) will return -3.

  12. type(A) will return the data type of A.

  13. max(A,B,C) will return the greatest numerical value in A,B,C. min(A,B,C) does exactly the opposite.

  14. round(A) will help us rounding A. i.e. 5.4 becomes 5 and 4.7 becomes 5.

  15. list1[0] A list can be indexed. [-1] will return the last element in the list, [-2] will return the second last element etc. [1:] will give all elements include and after position 1. [1:3] will return elements at position 1 & 2 but won’t include 3(左闭右开).

  16. list1[1] = new_value We can re-define an element value inside a list using this method.

  17. name = ["Oscar", "Jack" , "Michael"] number = [3,5,6,7] . If we want to combine these two lists, we could use name.append(number). This will return ["Oscar", "Jack" , "Michael",[3,5,6,7]]. In other words, adding a list into a list. We can also do name.extend(number). This will return ["Oscar", "Jack" , "Michael",3,5,6,7]

18 . Adding an element to a certain position in a list, we could use list1.insert(A,B). This means we want to put B(element) at position A.

  1. list1.remove(A) remove A(element) from the list.

  2. list1.pop() remove the last element from the list.

  3. list1.count(A) count how many times A(element) appears in a list.

  4. list1.sort() sort the list in default ascending order(numerical,alphabet). If we want to sort in descending order, use list1.sort(reverse = True).

  5. list1.reverse() This will reverse the order of the list.

  6. tuple1 = (A,B,C) use a small bracket to define. Its value can’t be changed once defined. We can have a list of tuples [(1,2),(3,4),(8,9)]

  7. if statement: the condition after if must be either True or False. Further Boolean algebra can be applied: or, and, nor etc.

  8. != means not equal in python

  9. dictionary1 = {key:value} In other words, key-value pairs are used to define a dictionary

  10. There are two ways of getting values in the dictionary. result=dictionary1[key]. This method will return an error if the key doesn’t exist. result=dictionary.get(key,default_value) If the key doesn’t exist, it will return None. We may also fill in a default value(optional) which will only be returned if the key doesn’t exist.

  11. While loop will keep running until the condition afterwards becomes False.

  12. Get out of one loop: break

  13. A string can be sliced. See the example.
